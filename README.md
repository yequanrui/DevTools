## Browser Settings

### 开启浏览器多线程下载

#### Chrome

地址栏输入并回车：`chrome://flags/#enable-parallel-downloading`

#### Edge(Chromium)

地址栏输入并回车：`edge://flags/#enable-parallel-downloading`

将默认的 `Default` 改成 `Enabled` 即可！然后点击 `Relaunch` 重启浏览器
